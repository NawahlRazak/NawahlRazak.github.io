L15 - Multiple Reflections

Demonstrates multiple reflections.

Multiple images with two plane mirrors. Two mirrors on large turntable. 6V battery. Only use 4V to minimise glare. Set mirrors at 90deg: object + 3 images (rotate) Set mirrors at 60deg: object + 5 images (rotate) Undo screws holding mirrors in position; mirrors can then be set almost parallel, facing each other. (Take care they don't fall over.) : object + infinite number of images, extending to infinity
